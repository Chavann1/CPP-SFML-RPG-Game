This is a C++ RPG game demo that was developed as a project for a University course.
The demo uses SFML 3.0 for window management and displaying the graphics.
The project was made in Visual Studio, and has some leftover VS files not added to gitignore.

Controls in the game are
W, A, S, D - movement
Z, X - actions

To build the project you need to have SFML 3.0 linked properly in Cmake or Visual Studio.
